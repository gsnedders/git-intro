Setup

Let git (and the world!) know who you are:

  • git config --global user.name 'John Smith'
  • git config --global user.email 'john@smith.me'

Fundementals

git has three main states files can reside in: committed, modified, and staged.

Starting Points

To start a new repository for the current directory, use:

git init

If you've created a repository on GitHub that you want to use, use:

git clone
